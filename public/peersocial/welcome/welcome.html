PeerSocial

A Social network for peers, by peers

Welcome

Welcome to PeerSocial, a P2P driven social network. The Goal is to have a friendly network with tools to create interactions. Think of me like im Every Social Media Network there is rolled into one. But there is a catch! You have to build it.

Intended Use

We intend this app to be decentralized. Allowing You and your Peers to control what you see and do without restrictions.

Goal Example use

Alice and Bob

Bob is a Developer and Alice is a peer. Bob has this idea to build a game, so he builds it in PeerSocial. Bob then publishes the code to his profile. Alice then can now run the published code on PeerSocial, allowing both Bob and Alice to interact within the game together.

This above example has a lot of holes to be filled. For example TRUST , a lot of people blindly trust a developer to be perfect and honest. And a Bad Peer may try to attack a Good Peer using the Example use.

What Comes with PeerSocial

PeerSocial comes with the basic stuff, Peer Management, Profile Management, and PeerApp Management Framework

With These 3 Things, A Peer who can write code can create a "PeerApp" to called PeerFeed. and PeerFeed can be a feed of posts that a peer follows.
